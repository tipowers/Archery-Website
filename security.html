Follow the steps below:

  1. Use Google Chrome (I will update this to support other browsers soon).
  2. Press F12 on your keyboard.
  3. The uppermost second tab from the left will say 'Console'. Click on it.
  4. Type the following statement: console.log("Web broswer name: " + navigator.appName);
  5. Press Enter
    • The response should be the name of the web browser you are using right now.
  6. Next type the following statement: console.log("Web broswer version: " + navigator.appVersion);
  7. Press Enter
    • The response should be the name of the web browser version you are currently using.
  8. Next type the following statement: console.log("Operating platform: " + navigator.platform);
  9. Press Enter
    • The response should be the name of your operating platform.
  10. Next type the following statement: console.log("User agent: " + navigator.userAgent);
  11. Press Enter
    • The response should be information about your browser, platform name, and compatibility.
  12. Next type the following statement: console.log("Geo Location: " + navigator.geolocation);
  13. Press Enter
    • The response should be information about your location in the world....should being the key word.
  14. Next type the following statement: console.log("Online: " + navigator.onLine);
  15. Press Enter
    • The response should return a true or false value. True being 'online' and false being 'offline'.

As you can see these 6 statements can reveal a lot about your web browser and its current status!
